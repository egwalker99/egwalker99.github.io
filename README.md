# README

## How to see what's changed

```
git status
```

## How to add files

```
git add .
```

## How to commit new things

```
git commit -am "my message (write something)"
```

## How to push things up to the site

```
git push origin master
```

## What's the log

```
git log
```

## How to pull in remote changes

```
git pull
```
